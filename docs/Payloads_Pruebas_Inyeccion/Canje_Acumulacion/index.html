Payload - Canje y Acumulación

CANJE Y ACUMULACION:

Proceso en el cual el cliente puede seleccionar 2 o más productos en una aplicación, los cuales puede canjear por productos a los cuales les alcance el puntaje y comprar el resto con su dinero; esta transacción se conoce como CANJE Y ACUMULACION.

PAYLOAD DE INYECCION MODALIDAD "CANJE Y ACUMULACION"

En el Payload que se muestra a continuación, se encuentran los datos para una transacción de canje Y acumulación donde en el mismo especifica dentro del apartado PRINCIPAL: 

"redeemPoints": true
"accumulatePoints": true

Estos datos especifican que el Payload contiene activo tanto el CANJE como la ACUMULACION
