Payloads para Pruebas de Inyección

Documentación del funcionamiento técnico para los procesos de fidelización dentro de las APPS.

📄️ Payload - Canje

Proceso en el cual el cliente puede canjear sus puntos totales acumulados por productos que estén dentro del alcance de puntos.

📄️ Payload - Acumulación

Proceso por el cual los clientes a través de sus compras acumulan puntos ($1 = 2pts)

📄️ Payload - Canje y Acumulación

Proceso en el cual el cliente puede seleccionar 2 o más productos en una aplicación, los cuales puede canjear por productos a los cuales les alcance el puntaje y comprar el resto con su dinero; esta transacción se conoce como CANJE Y ACUMULACION.
