Microservicios

El software está compuesto por pequeños servicios independientes, que se comunican entre sí a través de APIs bien definidas.

Programación reactiva

Se basa en la lógica de programación asíncrona y el uso de flujos de datos automatizados, para gestionar las actualizaciones de datos de los contenidos, cada vez que un usuario realiza una consulta.

Contenedores

Unidad de software, donde se empaquetan los ejecutables, el código binario, las bibliotecas y los archivos de configuración necesarios para que se pueda ejecutar en cualquier lugar.
